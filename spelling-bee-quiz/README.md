# 🐝 Spelling Bee Quiz App

A fun, kid-friendly spelling quiz app for the **Spelling Bee of Canada — Junior Category 2026**.

## Features

- 🔊 **Text-to-speech** — the app reads each word aloud automatically
- ✅ **Instant feedback** — tells you right away if you're correct or wrong
- 🔴 **Shows the correct spelling** if you get it wrong
- 💾 **Saves scores** across sessions using localStorage
- 🏆 **Leaderboard** — tracks top 10 scores by name
- 🎵 **Sound effects** — satisfying audio cues for correct and incorrect answers
- 📚 **All 400 words** from the official Junior SBOC 2026 study guide
- 🎲 **Random selection** — 10 different words every quiz

## How to Play

1. Enter your name on the home screen
2. Press **Start Quiz**
3. Listen to the word being spoken (it plays automatically)
4. Press 🔊 **Listen** again to hear it repeated
5. Type your spelling and press **Check** (or hit Enter)
6. Complete 10 words to see your score!

## Running the App

Simply open `index.html` in any modern web browser — no installation or internet connection required (after first load for fonts).

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/spelling-bee-quiz.git

# Open in browser
open index.html
```

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under *Source*, select **main branch / root**
4. Your app will be live at `https://YOUR_USERNAME.github.io/spelling-bee-quiz`

## Tech Stack

- Plain HTML, CSS, JavaScript — no frameworks or dependencies
- Web Speech API for text-to-speech
- Web Audio API for sound effects
- localStorage for score persistence

## Word List

All 400 words are sourced from the **Official SBOC Manual Word List, 2026 Edition — Junior Category**, as provided by the Spelling Bee of Canada.
